01. About Me

Hello! I'm Charles Nderitu, a passionate Software Engineer dedicated to building secure, scalable, and innovative digital solutions for real-world problems.

I specialize in full-stack software development, crafting everything from mobile applications to web platforms and backend APIs. My experience spans Android development with Kotlin and Jetpack Compose, web development with React and FastAPI, and database design with PostgreSQL.

04. Featured Projects

Garage Locator

A GPS-based Android app that helps drivers find nearby garages and auto repair shops in real time. Built with Kotlin and Jetpack Compose, it features live map integration, garage filtering by service type, booking functionality, and Firebase authentication — solving a practical mobility problem in the Kenyan market.

Kotlin Jetpack Compose Firebase Google Maps API GPS / Location

EduVote — Web3 Voting Platform

A blockchain-based voting application built for secure, transparent school and college elections. Leverages smart contracts on Ethereum to ensure votes are immutably recorded and tamper-proof. Features wallet-based authentication, real-time vote tallying, and an education-focused UI promoting civic engagement.

Solidity Hardhat Web3.js Ethereum React

Popote Listings — Property Marketplace

The React frontend for Popote Listings, a Kenyan property marketplace that enables users to browse, filter, and list properties for sale and rent. Features category-based browsing, search integration, image galleries, an admin panel, and WhatsApp inquiry flow. Connected to a FastAPI + PostgreSQL backend deployed on Render.

React FastAPI PostgreSQL Render TypeScript

Vera — AI Assistant App

A Kotlin-based AI personal assistant Android app powered by conversational AI. Vera handles natural language queries, task automation, and on-demand information retrieval with a clean Jetpack Compose UI. Designed with mobile-first UX and local-context intelligence for the Kenyan market.

Kotlin Jetpack Compose AI / NLP Firebase Android SDK

SmartFarm — Weather & Finance for Farmers

A Kotlin Android application serving as a smart weather and finance companion for farmers. Provides real-time weather forecasts, crop advisory based on conditions, market price tracking, and farm expense management — built to improve agricultural decision-making for smallholder farmers in Kenya.

Kotlin Jetpack Compose Weather API Firebase MVVM

SecureVault — Password Manager

A comprehensive password management Android app with AES-256 encryption, biometric authentication, and Firebase cloud backup with end-to-end encryption. Built with Kotlin and Jetpack Compose for a modern, secure user experience.
